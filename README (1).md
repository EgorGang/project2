## Описание проекта
Проект в рамках учебной программы - классный журнал, представленный в виде телеграм бота


## Пример использования
1) Чтобы самостоятельно запустить бота, потребуется создать его в Telegram при помощи @BotFather, скопировать его токен и вставить в соответствующую переменную "TOKEN" в модуле bot.py.
2) Далее установите все зависимости из файла requirements при помощи pip install
3) Теперь запустите основной файл проекта:
```bash
python main.py
```
4) Перейдите в Telegram к вашему боту (ссылка на него будет также у @BotFather) и запустите его. Станет доступен функционал из 8 кнопок:
- Добавить ученика - бот запросит имя и внесет ученика в журнал
- Удалить ученика - бот запросит имя и, если тот есть в журнале, удалит ученика из журнала
- Добавить оценку - бот запросит имя ученика и, если тот есть в журнале, запросит дату месяца и оценку, внеся ее в журнал
- Удалить оценку - бот запросит имя ученика и, если тот есть в журнале, запросит дату месяца и, если оценка там есть, то бот удалит ее из журнала
- Изменить оценку - бот запросит имя ученика и, если тот есть в журнале, запросит дату месяца и, если оценка там есть, запросит оценку, на которую надо ее изменить, после чего поменяет на новую 
- Информация об ученике - бот запросит имя ученика и, если тот есть в журнале, выдаст список оценок по датам и средний балл ученика
- Посмотреть журнал - бот покажет вам текущие данные журнала
- Экспорт журнала - бот запросит название для файла .xlsx и отправит вам его со всеми сохраненными данными
